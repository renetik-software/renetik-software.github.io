    How to record in Sampler Controller

    Choose controller preset

    Select recording length

    Long press on any sampler button

    Select device to record from

    Adjust input level

    Press button again to start recording

    After record finished it starts playing

    Long press recorded button again to open editor

    Select FX and click on empty effect slot button

    Choose effect form collection

    Choose effect preset or adjust equalizer params as you like

    Press save button to apply effect to recorded audio

    Choose with effects

    Now export audio

    Choose desired format

    Select desired bitrate

    Type desired file name

    And save or share wherever you like
